# LLM & Healthcare Research

We develop and evaluate **Large Language Model applications for healthcare** — from clinical assistants and diagnostic tools to standardized medical benchmarks.

Our work spans the full pipeline: dataset curation, model evaluation, on-premise deployment, and real-world clinical validation.

---

## Projects

| Repository | Description | Status |
|:-----------|:------------|:------:|
| [**ITAMed**](https://github.com/LM-Healthcare/ITAMed) | 📋 Comprehensive bilingual dataset (IT/EN) of 1,260 medical exam questions from the Italian National SSM (2017–2025), with specialty classification, image metadata, and multimodal content | [![Paper](https://img.shields.io/badge/Scientific%20Data-submitted-orange)](.) |
| [**LLM-EVAL-Education**](https://github.com/LM-Healthcare/LLM-EVAL-Education) | 📊 Benchmarking closed-source, open-weight, and quantized LLMs on the Italian Medical Specialization Exam (SSM 2020–2024) | [![Paper](https://img.shields.io/badge/Discover%20AI-submitted-orange)](.) |
| [**AURORA**](https://github.com/LM-Healthcare/AURORA) | 🏥 On-premise conversational clinical assistant for structured remote anamnesis (primary use case: Parkinson's Disease). FHIR-compatible, GDPR-native, fully offline | [![Status](https://img.shields.io/badge/status-active-green)](.) |
| [**LLM_DEMENTIA**](https://github.com/LM-Healthcare/LLM_DEMENTIA) | 🧠 LLM/SLM-based dementia diagnosis through incremental 3-step analysis: clinical data + RAG, plasma biomarkers, and CSF data | [![Status](https://img.shields.io/badge/status-active-green)](.) |

---

## Research Themes

- **Medical Benchmarking** — Evaluating LLMs on real clinical exams with rigorous methodology
- **Clinical NLP** — Structured information extraction from patient interactions
- **On-Premise AI** — Privacy-preserving, GDPR-compliant clinical deployments
- **Diagnostic AI** — LLM-assisted diagnosis with incremental clinical evidence
